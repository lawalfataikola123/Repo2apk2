## 🚀 What is Repo2APK?

Repo2APK is a powerful, no-code tool that lets you convert **any GitHub repository, website, or HTML file** into an Android APK file without writing a single line of code. Whether you have a Flutter app, React Native project, website, or HTML template - Repo2APK can turn it into a native Android app in minutes.

## ✨ Key Features

- **📦 GitHub Repo → APK**: Convert Flutter, React Native, Android, and other GitHub projects
- **🌐 Website → APK**: Wrap any website in a WebView Android app
- **📄 HTML → APK**: Bundle HTML/CSS/JS files into a native Android app
- **⚡ Fast Builds**: Most builds complete in ~2 minutes using GitHub Actions
- **🔐 Secure**: Your GitHub token stays in your browser - never sent to our servers
- **📱 Real APKs**: Generate genuine Android APK files ready for distribution
- **🎨 Customizable**: Set app names, icons, package names, and colors
- **💳 Credits System**: Remove ads and unlock premium features with one-time payments

## 🛠️ How It Works

1. **Get a GitHub Token**: Create a Personal Access Token with `repo` and `workflow` scopes
2. **Choose Your Input**: GitHub repo URL, website URL, or HTML file URL
3. **Configure Settings**: Set app name, icon, package name, etc.
4. **Build**: Click "Build APK" and watch the live console output
5. **Download**: Get your APK directly to your device
