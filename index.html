🔑 Setup GitHub Token

You need a GitHub Personal Access Token to trigger builds. It's free and takes 1 minute to create.

1. Go to github.com/settings/tokens/new
2. Note: "Repo2APK Builder"
3. Check: repo and workflow
4. Click "Generate token"
5. Copy and paste it below
