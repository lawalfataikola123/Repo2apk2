🔑 Setup GitHub Token

You need a GitHub Personal Access Token to trigger builds. Free and takes 1 minute!

1. Go to github.com/settings/tokens/new
2. Note: "Repo2APK Builder"
3. Check: repo and workflow
4. Click "Generate token" and copy it
