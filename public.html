📖 How to Use Repo2APK
Follow these steps to convert your projects into Android APKs
1
Get Your GitHub Token

A GitHub Personal Access Token (PAT) is required to trigger builds.

  1. Click the 🔑 Token button in the navigation bar
  2. Or go to github.com/settings/tokens/new
  3. Note: Repo2APK Builder
  4. Expiration: No expiration
  5. Select scopes: ✅ repo and ✅ workflow
  6. Click Generate token and copy it
  7. Paste it in the Token modal and click Save Token
2
Choose a Converter

Select the type of project you want to convert:

📦 Repo → APK
Convert Flutter, React Native, or Android projects from GitHub repositories
🌐 URL → APK
Wrap any website into a WebView Android app
📄 HTML → APK
Bundle HTML/CSS/JS files into a native Android app
❓ Frequently Asked Questions
How long does a build take?
Most builds complete in 2-3 minutes. Complex projects may take longer.
Is my GitHub token safe?
Yes! Your token is stored only in your browser's local storage. We never send it to our servers.
What types of projects can I build?
Flutter, React Native, and native Android projects from GitHub repos. Also any website or HTML file.
